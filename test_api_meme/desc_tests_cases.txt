1. Get memes

позитив
- статус код == 200
- проверка схемы json

негатив:
- ошибка авторизации
- ошибка 404

2. Get meme

позитив:
- статус код == 200
- проверка схемы json
- проверить данные созданного мема

негатив:
- ошибка авторизации
- ошибка получения мема которого не существует

3. Post meme

позитив:
- статус код == 200
- проверка схемы json
- проверить данные в ответе созданного мема

негатив:
- ошибка авторизации
- отправка данных в неверном формате - ошибка 400

4. Put meme

позитив:
- статус код == 200
- проверка схемы json
- проверить данные в ответе созданного мема
- отправка данных в неверном формате

негатив:
- ошибка авторизации
- ошибка 400
- ошибка прав на редактирование созданного мема
- ошибка редактирования мема которого не существует

5. delete meme

позитив:
- статус код == 200

негатив:
- ошибка авторизации
